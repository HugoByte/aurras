# Setup the SSB-Network For Test
This is a sample network for ssb for testing purpose. 

## Pre-requisite
- docker compose

## Steps for setup
- Start the Pub,consumer, and Producers

        ./ssb-up start 
- Create invite for pub

        ./ssb-up create-invite`
- Accept the invite

        ./ssb-up accept-invite`

- For Stop and clean

        ./ssb-up stop
